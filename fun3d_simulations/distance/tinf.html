T-infinity Distance Calculator¶

This distance calculator uses the ParallelDistanceCalculator from T-infinity to compute the distance from the wall. On the K cluster, the t-infinity/latest or t-infinity-openmpi/latest modules should to be loaded. If it is not available to you, use the default Refine Distance Calculator.
